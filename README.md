## Run Locally for Frontend Part

Clone the project

```bash or zsh
  git clone https://github.com/Faizan7012/triveous-assignment.git
```

Go to the project directory

```bash or zsh
  cd news-app
```

Install dependencies

```bash or zsh
  npm install
```

Start the server

```bash or zsh
  npm run start
```
